# 毛利优化插件 简易使用方法

注：仅适用于 NSudo_9.0_Preview1_PluginOnly_20210803_LegnaBrighost_v1.zip 的版本

## MoDefragMemory

### 插件功能

内存碎片整理

### 插件参数

无

## MoEnableMicrosoftUpdate

### 插件功能

开启 Microsoft Update

### 插件参数

无

## MoManageCompactOS

### 插件功能

CompactOS 管理

### 插件参数

- `/Query` 查询 CompactOS 状态
- `/Enable` 启用 CompactOS
- `/Disable` 禁用 CompactOS

## MoPurgeChromiumCache

### 插件功能

Chromium 内核缓存清理

### 插件参数

- `/Scan` 扫描
- `/Purge` 清理
